# 并发编程

## 乐观锁

* 每次假设没有改变

```groovy
def user = User.get(1)
user.username = "张三"
user.save(flush: true)
```

## 悲观锁
* 每次加锁，操作完成后释放

```groovy
def user = User.lock(1) 
// 或者user.lock()
user.username = "张三"
user.save(flush: true)
```

## 同步代码块

```groovy
def update() {
    synchronized (SiteInfoService.class) {
        SiteInfo site = SiteInfo.all.find {
            it.id > 0
        }
        if (site) {
            SiteInfo.withNewTransaction {
                site.pvCount += 1
                site.save(flush: true)
            }
            return site
        }
    }
    return null
}
```
