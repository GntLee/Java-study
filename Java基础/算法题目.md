# 算法题

* 有一条长长的阶梯，如果每步跨2阶，那么最后剩1阶；如果每步跨3阶，那么最后剩2阶；如果每步跨5阶，那么最后剩4阶；
如果每步跨6阶，最后剩5阶；只有当每步跨7阶时，最后才正好走完，一阶也不剩。请问这条阶梯至少有多少阶？

```java
public static void main(String[] args) {
    int i = 1;
    boolean flag = true;
    while (flag) {
        if (i % 2 == 1 && i % 3 == 2 && i % 5 == 4 && i % 6 == 5 && i % 7 == 0) {
            System.out.printf("i = %d \n",i);
            flag = false;
        }
        i++;
    }
}

```
