# 操作

* 常用运算

|操作|方法|
|--|--|
|a + b| a.plus(b)|
|a - b| a.minus(b)|
|a * b| a.multiply(b)|
|a / b| a.divide(b)|
|a++ or ++a| a.next()|
|a-- or --a| a.previous()|
|a[b]| a.getAt(b)|
|a[b] = c| a.putAt(b, c)|
|a << b| a.leftShift(b)|

* 比较
|操作|方法|
|--|--|
|a == b| a.equals(b)|
|a != b| ! a.equals(b)|
|a <=>| b a.compareTo(b)|
|a > b| a.compareTo(b) > 0|
|a >= b| a.compareTo(b) >= 0|
|a < b| a.compareTo(b) < 0|
|a <= b| a.compareTo(b) <= 0|
